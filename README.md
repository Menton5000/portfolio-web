# Portfolio

Mi portfolio personal construido con **Astro** y **Tailwind CSS**. Basado en el excelente tutorial de [@midudev](https://github.com/midudev).

## Tecnologías utilizadas

*   **Astro** - Framework web para la construcción de sitios rápidos orientados al contenido.
*   **Tailwind CSS 4** - Framework CSS Utility-first para un diseño rápido y moderno.
*   **Bun** - Gestor de paquetes y entorno de ejecución rápido.

## Estructura del Proyecto

Dentro de su proyecto Astro, encontrará los siguientes directorios y archivos:

-   `src/components/`: Componentes reutilizables de Astro.
-   `src/layouts/`: Plantillas comunes para las páginas.
-   `src/pages/`: Páginas del sitio (rutas). Cada archivo `.astro` aquí se convierte en una página.
-   `public/`: Recursos estáticos como imágenes y fuentes.

## Cómo empezar

Para ejecutar este proyecto localmente, asegúrate de tener [Bun](https://bun.sh/) instalado.

1.  **Instalar dependencias:**
    ```bash
    bun install
    ```

2.  **Iniciar el servidor de desarrollo:**
    ```bash
    bun dev
    ```

3.  **Compilar para producción:**
    ```bash
    bun run build
    ```
